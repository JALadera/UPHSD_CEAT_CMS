## 🛠️ Installation Guide for Cloners

After someone clones from GitHub:

```bash
# 1. Install dependencies
composer install
npm install

# 2. Setup environment
cp .env.example .env
php artisan key:generate

# 3. Database setup
php artisan migrate
php artisan db:seed

# 4. Build assets
npm run build

# 5. Start server
php artisan serve

# 6. Access application
# Admin: http://localhost:8000/admin
# Public: http://localhost:8000
```
